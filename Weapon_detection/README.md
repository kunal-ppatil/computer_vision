# Weapon Detection Project 

This project detects guns in live video feeds using **OpenCV** and **Haar cascade classifiers**.

## Contents

- `weapon_detection.ipynb` : Jupyter notebook with code to capture video, detect guns, and display results.
- `gun_cascade.xml` : Haar cascade classifier used for gun detection.
- `README.md` : Project description.

## Features

- Real-time video capture from webcam
- Gun detection using Haar cascades
- Bounding boxes around detected objects
- Logs detection events in console
